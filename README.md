# RARA ARAR AAA

Skákal pes,
přes oves!
Přes zelenou louku,
šel za nim myslivec,
to byste neřekly a von měl péro na klobouku.
Pejsku náš co děláš žes tak vesel stále?
Řek bych Vám, ale sám nevim.
Hop a skákal dále.
