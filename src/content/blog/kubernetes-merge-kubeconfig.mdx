1. Merge the Configs

```bash
KUBECONFIG=<your_first_kubeconfig>:<your_second_kubeconfig> \
kubectl config view --flatten > ~/.kube/config
```

2. You can then verify the merge by using something like `kubectx`
```bash
kubectx
```
