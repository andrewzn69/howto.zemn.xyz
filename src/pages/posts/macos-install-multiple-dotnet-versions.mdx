1. Add the Microsoft package feed to your system
```bash
brew tap isen-ng/dotnet-sdk-versions
```
2. Install the desired version of the .NET SDK
```bash
brew install --cask dotnet-sdk-6-0-400
brew install --cask dotnet-sdk-7-0-400
```
3. Verify the installation
```bash
dotnet --list-sdks
```

This way you can use multiple versions of the .NET SDK in parallel.

In your .NET code you can specify the desired version in the global.json file:
```json
{
  "sdk": {
    "version": "7.0.400"
  }
}
```
