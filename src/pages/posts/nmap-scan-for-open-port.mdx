1. Install nmap

```bash
# Debian/Ubuntu
sudo apt-get install nmap

# Arch
sudo pacman -S nmap

# RHEL/CentOS/Fedora
sudo dnf install nmap
```

2. Scan for open port

```bash
nmap -p <port> -sC <ip>
```
