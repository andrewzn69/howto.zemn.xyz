1. Drop all ICMP

```bash
firewall-cmd --set-target=DROP --zone=public --permanent
firewall-cmd --zone=public --remove-icmp-block={echo-request,echo-reply,timestamp-request,timestamp-reply} --permanent
```

2. Reload the firewall

```bash
firewall-cmd --reload
```
